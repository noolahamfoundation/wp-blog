Thank you for choosing Evanescence WordPress theme version 1.6. Visit http://srinig.com/wordpress/themes/evanescence/ for updates and newer versions of the theme.

Features
* Fluid width
* Two columns
* Right hand sidebar
* Supports sidebar widgets, post tags and gravatars
* Compatible upto WordPress version 2.6
* XHTML and CSS validated

Installation:
* Extract the evanescence.zip file in a convenient location
* Copy the 'evanescence' folder to '/wp-content/themes/' directory in your web server.
* Log in to the WordPress Administration Panels
* Select the 'Design' menu.
* Select the Evanescence theme from the available list of themes to preview
* Activate the theme.
* Click 'View Site' at the top of the Panel's screen. Your selection should immediately become active.
